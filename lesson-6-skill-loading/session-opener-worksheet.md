# Session Opener Worksheet
## Foundations of Claude — Week 2 Exercise

### What This Is

Every time you start a writing session with Claude, you need to tell it exactly what you're working on. Claude can see your skill files exist, but it doesn't load them until you ask. This worksheet helps you build the habit of opening every session correctly.

Fill out one template for each active writing project. Keep these somewhere you can copy-paste from — a notes app, a Google Doc, a text file on your desktop. You'll use them every single time.

---

### The Session Opener Template

Copy this block and paste it at the start of every writing session:

```
Pen name: [Your pen name for this project]
Project: [Book title] — [new draft / continuing]
Load skill: [The exact skill name to trigger]
Context: [What chapter or scene you're working on. Where the story left off.
          POV character's current emotional state. Key details Claude needs.]
Voice notes: [Anything specific to THIS session that overrides or adjusts the
             default skill — tone shifts, heat level changes, character notes,
             pacing instructions.]
```

---

### Field-by-Field Guide

**Pen name** — Which author identity is writing this? Don't skip this even if you think it's obvious. Claude doesn't carry this over from your last session.

**Project** — The book title and whether this is a brand new draft or a continuation. If continuing, Claude knows to ask about or reference previous chapters.

**Load skill** — This is the critical line. This is what makes Claude actually open your voice skill file and read the instructions. Without this, your skill is just a title on a shelf. Use the exact skill name (e.g., "kelsey-croft" not "Kelsey Croft's skill" or "my vampire book skill").

**Context** — Think of this as your "previously on" recap. Claude can't scroll back through your manuscript. Give it the runway it needs: what just happened, where the character is emotionally, any plot threads that are live in this moment.

**Voice notes** — Session-specific adjustments. Maybe your skill says heat level 4, but this scene needs restraint. Maybe your character is normally bold but right now she's scared. Maybe you want shorter sentences for a tense chapter. This is your fine-tuning dial.

### Bonus: Mid-Session Switch Template

When you change projects or pen names during a session, paste this reset before your next request:

```
Switching projects. New Session Opener:
Pen name:
Project:
Load skill:
Context:


Voice notes:


```

This tells Claude to drop the previous voice and load a fresh one. Without this reset, you'll get the old pen name's voice applied to the new project.

---

### Common Mistakes to Avoid

**"Claude already knows what I'm working on."** — It doesn't. Every chat session starts fresh. Your preferences and CLAUDE.md carry over, but your project context and voice skills do not.

**"I mentioned the pen name in passing, that's enough."** — Mentioning a pen name in conversation is not the same as triggering the skill. The "Load skill:" line is what actually opens the file.

**"I'll just say 'write the next chapter.'"** — Next chapter of what? In whose voice? At what heat level? From which POV character's perspective? Claude will produce something — but it won't be calibrated to your brand.

**"I switched pen names but didn't reset."** — If you go from writing Kelsey Croft to writing Sage Kennedy without a new Session Opener, you'll get Kelsey's dark paranormal voice writing a romcom hockey scene. The output will feel off and you won't immediately know why.
