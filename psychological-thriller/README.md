# Psychological Thriller Writing Skill

**A product of [The Invisible Pen](https://theinvisiblepen.com) | $47 on [Payhip](https://payhip.com/theinvisiblepen)**

A comprehensive AI-assisted writing system for psychological thriller authors. This Claude Code plugin gives you a genre-trained writing partner that understands confined-space thrillers, domestic suspense, and surreal horror at a craft level.

## What's Included

- **SKILL.md** -- Full genre writing system covering voice, pacing, POV, protagonist/antagonist archetypes, scene structure, and hard limits for psychological thrillers
- **Scene Checklist** -- 15-point verification checklist with detailed explanations for every scene you write
- **Pacing Guide** -- Chapter-level and book-level pacing rules, tension curves, and structural templates
- **thriller-check command** -- One-command scene evaluation that scores your work against genre standards and gives actionable feedback

## Sub-genres Covered

- Confined-space thrillers (architectural horror, escape narratives)
- Domestic/relational suspense (marriage-as-cage, institutional control)
- Surreal psychological horror (dream logic grounded in emotional truth)

## Installation

1. Copy the `psychological-thriller` folder into your project's `.claude/plugins/` directory
2. The skill and command will be available automatically in Claude Code
3. Use the `thriller-check` command to evaluate any scene against genre standards

## Usage

Once installed, Claude will automatically apply psychological thriller genre conventions when you're working on thriller content. You can also:

- Ask Claude to draft scenes using the skill's voice and pacing rules
- Run `/thriller-check` on any scene for a detailed genre-standards evaluation
- Reference the pacing guide when planning chapter structure
- Use the scene checklist as a pre-submission quality gate

- Claude Code (Claude's official CLI)
