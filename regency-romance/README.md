# Regency Romance Writing Skill

**A product of [The Invisible Pen](https://theinvisiblepen.com) | $47 on [Payhip](https://payhip.com/theinvisiblepen)**

A comprehensive AI-assisted writing system for Regency romance authors. This Claude Code plugin gives you a genre-trained writing partner that understands period-accurate language, social conventions, courtship rules, and the art of Jane Austen-adjacent romantic tension.

## What's Included

- **SKILL.md** -- Full genre writing system covering voice, language patterns, historical accuracy, titles and forms of address, courtship conventions, heat level guidance, trope library, and scene structure
- **Period Language Guide** -- Complete dos and don'ts for Regency-era dialogue and narration, including word substitutions, anachronisms to avoid, and sentence structure patterns
- **Scene Checklist** -- Detailed verification checklist covering language, historical accuracy, romantic tension, and craft fundamentals
- **regency-check command** -- One-command scene evaluation that scores your work against Regency genre standards

## What You'll Write Better

- Traditional Regency romance (clean to warm)
- Modern Regency romance (warm to steamy)
- Regency romantic comedy
- Regency mystery/suspense with romance
- Any historical romance set in the 1795-1830 period

## Installation

1. Copy the `regency-romance` folder into your project's `.claude/plugins/` directory
2. The skill and command will be available automatically in Claude Code
3. Use the `regency-check` command to evaluate any scene against genre standards

## Usage

Once installed, Claude will automatically apply Regency romance conventions when you're working on period content. You can also:

- Ask Claude to draft scenes with period-accurate language and social rules
- Run `/regency-check` on any scene for a detailed genre-standards evaluation
- Reference the period language guide when writing dialogue
- Use the scene checklist as a pre-submission quality gate

## Requirements

- Claude Code (Claude's official CLI)
- An active Claude subscription
