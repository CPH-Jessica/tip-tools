# Pen Name Launcher

**A product of [The Invisible Pen](https://theinvisiblepen.com) — AI-powered author education by Jessica Waldron**

Launch your pen name the right way — with a complete 10-phase system that walks you through every decision, every platform, and every cost.

## What's Included

### Claude Plugin (Install in Cowork)
An installable Claude skill that guides you through all 10 phases of launching a pen name:

1. Pen Name Identity
2. Brand & Visual Identity
3. Newsletter Setup
4. Swap Platform Accounts
5. Reader Magnet Creation
6. Cover Design
7. KDP Publishing
8. Draft2Digital (Going Wide)
9. Marketing & Ads
10. Launch Checklist

### Self-Paced Class (6 Lessons)
Complete course materials including:
- **Video scripts** — Ready for AI avatar recording (15-20 min each)
- **Slide descriptions** — 15-23 slides per lesson
- **Printable worksheets** — Fill-in workbooks for each lesson
- **Interactive HTML worksheets** — Fill out directly in your browser with auto-save

### Slash Command
- `/launch-checklist` — Check your progress across all 10 phases

## Installation

1. Download or clone this directory
2. Install in Claude Cowork as a plugin
3. Say "help me launch my pen name" to get started

## Requirements

- Claude Pro subscription ($20/month) recommended
- Accounts on KDP, Mailerlite (or Kit/Mailchimp), and BookFunnel
