## CANVA DESIGN CHECKLIST

As you design your cover in Canva:

- [ ] Used pre-made book cover template (dimensions correct)
- [ ] Uploaded AI image onto template
- [ ] Title placed in high-contrast area
- [ ] Author name placed and readable
- [ ] Font choices tested at thumbnail size
- [ ] No more than 2 fonts used
- [ ] Text is bold and clear (not script)
- [ ] Semi-transparent overlay added (if needed for text)
- [ ] Colors match genre expectations
- [ ] Composition balanced and professional-looking
- [ ] Thumbnail test passed (genre is immediately clear)
- [ ] No design elements are cut off at edges
- [ ] Color scheme uses 3-4 main colors max
