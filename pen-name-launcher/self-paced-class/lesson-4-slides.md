### Slide 11: CANVA DESIGN FUNDAMENTALS
**Your Canva Workflow:**
1. Start with pre-made book cover template (dimensions already correct)
2. Upload AI-generated image
3. Add title + author name with high contrast
4. Adjust composition (crop, move, overlay)
5. Test at thumbnail size
6. Refine typography and colors

Visual: Canva interface with cover in progress

---

### Slide 12: CANVA TEXT BEST PRACTICES
**DO:**
- Use bold, sans-serif fonts at large size
- High contrast (white text on dark, dark on light)
- Limit to 2 fonts maximum
- Test at thumbnail size

**DON'T:**
- Use script fonts (impossible to read small)
- Low contrast text
- More than 3 fonts
- Assume it works at thumbnail without testing

Visual: Examples of good vs bad cover typography

### Slide 14: CANVA PRICING
**Free vs Pro:**
- **Free**: Basic book cover templates, limited design assets, no resize tool
- **Pro ($15/mo)**: Unlimited assets, resize covers instantly, design to multiple formats

Text: "Pro pays for itself if you're making more than 2 covers/month"

### Slide 15: BOOK BRUSH FINISHING
**What Book Brush Does:**
- Ensures ebook cover meets Amazon specs (1600x2560px, RGB, right DPI)
- Creates print-ready cover with spine (calculates width based on page count)
- Generates square social graphics automatically
- One upload = multiple formats

Visual: Book Brush dashboard showing format options

Visual: Shows one cover converted to three formats

### Slide 17: WHEN TO HIRE A DESIGNER
**DIY Makes Sense If:**
- You're comfortable with design tools
- You're launching 1-3 books initially
- Budget is tight
- Timeline is flexible

**Hire a Designer If:**
- You're launching 10+ books with one pen name
- You have budget ($75-500/cover)
- You want visual consistency immediately
- Timeline is urgent
