## USAGE NOTES

### For Avatar Video Production
Use the `*-script.md` files with avatar video tools (Synthesia, Colossyan, etc.). Each script includes:
- [PAUSE] markers for natural breaks
- [SHOW ON SCREEN: description] markers for visual cues and screen elements
- [TRANSITION] markers between sections
- Clear pacing (15-20 minutes per lesson)

### For Worksheet Delivery
Use the `*-worksheet.md` files for student workbooks. Students fill these in while watching the video or after. Include with course materials in your platform (Kajabi, Thinkific, Teachable, etc.).

### For Slide Design
Use the `*-slides.md` files as specifications for visual design. Each slide includes:
- Title, subtitle, key content
- Visual description (what should be shown)
- Design style notes (tone, emphasis)
- Total 15-20 slides per lesson

**Recommendation:** Use design tool like Canva to create polished slides from these descriptions. Maintain consistent brand identity (colors, fonts from Lesson 1 exercises).
