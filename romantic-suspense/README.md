# Romantic Suspense Writing Skill

**A product of [The Invisible Pen](https://theinvisiblepen.com) | $47 on [Payhip](https://payhip.com/theinvisiblepen)**

A comprehensive AI-assisted writing system for romantic suspense authors. This Claude Code plugin gives you a genre-trained writing partner that understands the delicate balance of romance and suspense arcs, dual-POV pacing, and the art of weaving danger with desire.

## What's Included

- **SKILL.md** -- Full genre writing system covering dual-arc structure, POV guidance, heat levels, protagonist/antagonist archetypes, suspense mechanics, romance-suspense balance rules, and scene structure
- **Dual-Arc Pacing Guide** -- How to interweave romance and suspense arcs so neither overwhelms the other, with chapter-level templates and act structure
- **Scene Checklist** -- Detailed verification checklist covering both romance and suspense elements in every scene
- **suspense-check command** -- One-command scene evaluation that scores your work against romantic suspense genre standards

## Sub-genres Covered

- Romantic thriller (heavier suspense weight)
- Romantic mystery (whodunit + romance)
- Military/special ops romantic suspense
- Small-town romantic suspense
- Romantic suspense with paranormal elements

## Installation

1. Copy the `romantic-suspense` folder into your project's `.claude/plugins/` directory
2. The skill and command will be available automatically in Claude Code
3. Use the `suspense-check` command to evaluate any scene against genre standards

## Usage

Once installed, Claude will automatically apply romantic suspense genre conventions when you're working on romantic suspense content. You can also:

- Ask Claude to draft scenes that balance both arcs effectively
- Run `/suspense-check` on any scene for a detailed genre-standards evaluation
- Reference the dual-arc pacing guide when plotting your novel
- Use the scene checklist as a pre-submission quality gate

- Claude Code (Claude's official CLI)
